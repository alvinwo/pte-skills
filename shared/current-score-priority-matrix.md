# Current PTE Score Priority Matrix

This file is an updated **practical score matrix** for the current PTE Academic format.

## Important caveat
Pearson's current official materials **do not publish a public percentage weighting table** like some older prep charts circulating online.

However, the user has now provided a newer third-party matrix image labeled:
- **猩际 PTE**
- **PTE 内容题型 & 分数信息表 V4.1 (25.08.07 开始)**

That matrix is **not an official Pearson document**, but it appears to be a more current operational estimate than older 2022-era charts because it:
- includes the newer task types **Summarize Group Discussion (SGD)** and **Respond to a Situation (RTS)**
- normalizes weights to 100% within each communicative skill
- reflects the current 22-question-type structure

So this file now uses a **two-layer approach**:
1. **Official Pearson structure and scoring traits** as the hard source of truth
2. **User-shared V4.1 matrix** as the best current practical weighting reference, with caution that real exams may vary by item count

What Pearson does publish officially:
- current question types
- approximate number of questions
- section timings
- communicative skills assessed
- scoring traits and maximum trait scores for many tasks

So this file is designed to be the **latest defensible matrix** we can use for coaching users:
- official structure where available
- official scoring-trait maxima where available
- practical priority guidance for target-score planning

## How to use this matrix
When helping a learner:
1. Ask for target scores:
   - overall target
   - speaking target
   - writing target
   - reading target
   - listening target
2. Compare weak skills with the question types that feed those skills.
3. Prioritize tasks that are both:
   - frequent
   - heavily scored or trait-rich
4. Build the study plan around those tasks first.

- **Describe Image** is the largest Speaking contributor in this matrix.
- **Repeat Sentence** remains extremely important for both Speaking and Listening.
- **Summarize Group Discussion** is now a major task for both Speaking and Listening.
- **Respond to a Situation** matters enough that it should not be treated as a low-priority side task.
- **Write from Dictation** remains critical for Writing and strong for Listening.
- **Reading multiple choice tasks remain relatively low-yield** compared with blanks and integrated tasks.

### Official raw-score observations for key Speaking & Writing tasks
These are not Pearson percentages. They are rough **max trait-point ranges** derived from the official score guide.

- Repeat Sentence: about **130–156** raw trait points across the test
- Describe Image: about **80–96** raw trait points across the test
- Retell Lecture: about **32–48** raw trait points across the test
- Summarize Group Discussion: about **32–48** raw trait points across the test
- Respond to a Situation: about **32–48** raw trait points across the test
- Summarize Written Text: about **18** raw trait points across the test
- Write Essay: about **26** raw trait points across the test

### Practical takeaway for target scores
For most learners targeting high Speaking / Listening scores, the first tier is usually:
1. Repeat Sentence
2. Read Aloud
3. Describe Image
4. Retell Lecture / Summarize Group Discussion / Respond to a Situation

For Writing targets, the first tier is usually:
1. Write from Dictation
2. Fill in the Blanks (Type In)
3. Fill in the Blanks (Drag and Drop) where relevant to writing integration
4. Write Essay / Summarize Written Text / Summarize Spoken Text

## Part 2: Reading

| Question type | Official count | Official task time | Skills scored | Official scoring notes | Practical priority |
|---|---:|---|---|---|---|
| Fill in the Blanks (Dropdown) | 5-6 | section timed | Reading | 1 point per correct blank | High |
| Multiple Choice, Multiple Answers | 2-3 | section timed | Reading | +1 correct, -1 incorrect choice | Low-Medium |
| Reorder Paragraph | 2-3 | section timed | Reading | 1 point per correct adjacent pair | Medium |
| Fill in the Blanks (Drag and Drop) | 4-5 | section timed | Reading | 1 point per correct blank | High |
| Multiple Choice, Single Answer | 2-3 | section timed | Reading | correct / incorrect | Low |

### Practical takeaway for Reading targets
If the learner needs Reading improvement, prioritize:
1. Fill in the Blanks (Dropdown)
2. Fill in the Blanks (Drag and Drop)
3. Reorder Paragraph
Then do MCQ work after the high-yield tasks are under control.

## Updated coaching rule for Hermes skills
Before recommending drills, ask:
- What is your target overall PTE score?
- What are your target scores for Speaking, Writing, Reading, and Listening?
- Which score is currently the weakest?
- Do you want a safe improvement plan or a fastest-score-gain plan?

Then route learners to the highest-yield question types for their weak skills.

For scenario-specific planning, also consult `shared/scenario-based-study-plans.md` so the agent can adjust recommendations for:
- lower-threshold student visa users
- all four communicative skills at 65+ users who need balanced section protection
- all four communicative skills at 79+ users who need stronger consistency and fewer point leaks

## Final note
