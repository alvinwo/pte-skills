# Current PTE Score Priority Matrix

This file is an updated **practical score matrix** for the current PTE Academic format.

## Important caveat
Pearson's current official materials **do not publish a public percentage weighting table** like some older prep charts circulating online.

What Pearson does publish officially:
- current question types
- approximate number of questions
- section timings
- communicative skills assessed
- scoring traits and maximum trait scores for many tasks

So this file is designed to be the **latest defensible matrix** we can use for coaching users:
- official structure where available
- official scoring-trait maxima where available
- practical priority guidance for target-score planning

## Official sources checked
- `https://www.pearsonpte.com/pte-academic/test-format`
- `https://www.pearsonpte.com/pte-academic/test-format/speaking-writing`
- `https://www.pearsonpte.com/pte-academic/test-format/reading`
- `https://www.pearsonpte.com/pte-academic/test-format/listening`
- `https://www.pearsonpte.com/pte-academic/scoring`
- `PTE Academic Test Taker Score Guide` PDF linked from Pearson scoring page

## How to use this matrix
When helping a learner:
1. Ask for target scores:
   - overall target
   - speaking target
   - writing target
   - reading target
   - listening target
2. Compare weak skills with the question types that feed those skills.
3. Prioritize tasks that are both:
   - frequent
   - heavily scored or trait-rich
4. Build the study plan around those tasks first.

## Part 1: Speaking & Writing

| Question type | Official count | Official task time | Skills scored | Official scoring traits / notes | Practical priority |
|---|---:|---|---|---|---|
| Personal Introduction *(not scored)* | 1 | 25s prep + 30s speak | Not scored | Familiarization only | None |
| Read Aloud | 6-7 | ~35–40s prep + 40s speak | Speaking | Content + Pronunciation + Oral Fluency; content max depends on prompt length | High for Speaking |
| Repeat Sentence | 10-12 | 15s speak | Listening + Speaking | Content 0-3, Pronunciation 0-5, Oral Fluency 0-5 | Very High |
| Describe Image | 5-6 | 25s prep + 40s speak | Speaking | Content 0-6, Pronunciation 0-5, Oral Fluency 0-5 | High |
| Retell Lecture | 2-3 | 10s prep + 40s speak | Listening + Speaking | Content 0-6, Pronunciation 0-5, Oral Fluency 0-5 | Medium-High |
| Answer Short Question | 5-6 | 10s speak | Listening | Vocabulary 0-1 | Low-Medium |
| Summarize Group Discussion | 2-3 | 10s prep + 2 mins speak | Listening + Speaking | Content 0-6, Pronunciation 0-5, Oral Fluency 0-5 | Medium-High |
| Respond to a Situation | 2-3 | 10s prep + 40s speak | Speaking | Content 0-6, Pronunciation 0-5, Oral Fluency 0-5 | Medium-High |
| Summarize Written Text | 2 | 10 mins each | Reading + Writing | Content 0-4, Form 0-1, Grammar 0-2, Vocabulary 0-2 | Medium |
| Write Essay | 1 | 20 mins | Writing | Content 0-6, Form 0-2, Development/Structure/Coherence 0-6, Grammar 0-2, General Linguistic Range 0-6, Vocabulary Range 0-2, Spelling 0-2 | Medium |

### Official raw-score observations for key Speaking & Writing tasks
These are not Pearson percentages. They are rough **max trait-point ranges** derived from the official score guide.

- Repeat Sentence: about **130–156** raw trait points across the test
- Describe Image: about **80–96** raw trait points across the test
- Retell Lecture: about **32–48** raw trait points across the test
- Summarize Group Discussion: about **32–48** raw trait points across the test
- Respond to a Situation: about **32–48** raw trait points across the test
- Summarize Written Text: about **18** raw trait points across the test
- Write Essay: about **26** raw trait points across the test

### Practical takeaway for target scores
For most learners targeting high Speaking / Listening scores, the first tier is usually:
1. Repeat Sentence
2. Read Aloud
3. Describe Image
4. Retell Lecture / Summarize Group Discussion / Respond to a Situation

For Writing targets, the first tier is usually:
1. Write from Dictation
2. Fill in the Blanks (Type In)
3. Fill in the Blanks (Drag and Drop) where relevant to writing integration
4. Write Essay / Summarize Written Text / Summarize Spoken Text

## Part 2: Reading

| Question type | Official count | Official task time | Skills scored | Official scoring notes | Practical priority |
|---|---:|---|---|---|---|
| Fill in the Blanks (Dropdown) | 5-6 | section timed | Reading | 1 point per correct blank | High |
| Multiple Choice, Multiple Answers | 2-3 | section timed | Reading | +1 correct, -1 incorrect choice | Low-Medium |
| Reorder Paragraph | 2-3 | section timed | Reading | 1 point per correct adjacent pair | Medium |
| Fill in the Blanks (Drag and Drop) | 4-5 | section timed | Reading | 1 point per correct blank | High |
| Multiple Choice, Single Answer | 2-3 | section timed | Reading | correct / incorrect | Low |

### Practical takeaway for Reading targets
If the learner needs Reading improvement, prioritize:
1. Fill in the Blanks (Dropdown)
2. Fill in the Blanks (Drag and Drop)
3. Reorder Paragraph
Then do MCQ work after the high-yield tasks are under control.

| Question type | Official count | Official task time | Skills scored | Official scoring notes | Practical priority |
|---|---:|---|---|---|---|

| Highlight Correct Summary | 2-3 | section timed | Listening + Reading | correct / incorrect | Low-Medium |
| Multiple Choice, Single Answer | 2-3 | section timed | Listening | correct / incorrect | Low |
| Select Missing Word | 1-2 | section timed | Listening | correct / incorrect | Low |
| Highlight Incorrect Words | 2-3 | section timed | Listening + Reading | +1 correct, -1 incorrect | High |
| Write from Dictation | 3-4 | section timed | Listening + Writing | 1 point per correct word spelled correctly | Very High |

### Practical takeaway for Listening targets
If the learner needs Listening improvement, prioritize:
1. Write from Dictation
2. Repeat Sentence
3. Highlight Incorrect Words
4. Fill in the Blanks (Type In)
5. Retell Lecture / Summarize Group Discussion / Summarize Spoken Text

## Updated coaching rule for Hermes skills
Before recommending drills, ask:
- What is your target overall PTE score?
- What are your target scores for Speaking, Writing, Reading, and Listening?
- Which score is currently the weakest?
- Do you want a safe improvement plan or a fastest-score-gain plan?

Then route learners to the highest-yield question types for their weak skills.

## Recommended default priority map
### If Speaking is weak
- Repeat Sentence
- Read Aloud
- Describe Image
- Respond to a Situation
- Summarize Group Discussion
- Retell Lecture

### If Listening is weak
- Write from Dictation
- Repeat Sentence
- Highlight Incorrect Words
- Fill in the Blanks (Type In)
- Summarize Group Discussion
- Retell Lecture

### If Reading is weak
- Fill in the Blanks (Dropdown)
- Fill in the Blanks (Drag and Drop)
- Reorder Paragraph
- Summarize Written Text

### If Writing is weak
- Write from Dictation
- Fill in the Blanks (Type In)
- Write Essay
- Summarize Written Text
- Summarize Spoken Text

## Final note
Older percentage charts from 2022 or earlier may still be directionally useful, but they are not sufficient as the latest source of truth because:
- question types have changed
- counts have changed
- the official score guide now includes newer tasks
- Pearson does not currently publish a single public percentage table for the full modern test

So for this skill pack, this matrix should be treated as the current operational baseline until a newer official scoring breakdown becomes available.
