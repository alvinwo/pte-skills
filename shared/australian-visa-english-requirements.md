# Australian Visa English Requirements

This file stores the current English-level requirements relevant to migration-oriented PTE learners.

## Purpose
Use this page when a learner is studying for an Australian visa requirement rather than for a generic PTE target.

The agent should:
1. ask the learner which English level they need
2. ask the visa subclass if known
3. use the thresholds below as the current working reference
4. validate the requirement again at runtime against the official Department of Home Affairs pages before giving final advice

## Runtime validation rule
These requirements can change.

Before giving final migration advice, re-check the official source pages:
- main page: https://immi.homeaffairs.gov.au/help-support/meeting-our-requirements/english-language

Use them as the repo's current baseline, then validate live.

## Current note from Home Affairs
Home Affairs states that the approved English tests changed on 7 August 2025.
For PTE Academic, the test keeps the same name as the previous version, but the accepted scores are different.

## PTE Academic thresholds

| Level | PTE requirement | Validity note |
| --- | --- | --- |
| Functional English | overall score at least 24 | generally within 12 months before visa application, depending on visa subclass |
| Vocational English | listening 33, reading 36, writing 29, speaking 24 | generally up to 3 years, depending on visa subclass |
| Competent English | listening 47, reading 48, writing 51, speaking 54 | generally up to 3 years, depending on visa subclass |
| Proficient English | listening 58, reading 59, writing 69, speaking 76 | generally up to 3 years, depending on visa subclass |
| Superior English | listening 69, reading 70, writing 85, speaking 88 | generally up to 3 years, depending on visa subclass |

## Other accepted evidence paths
Depending on the level and visa subclass, Home Affairs may also accept:
- a passport from an eligible English-speaking country
- qualifying study completed in English
- other approved English tests

The agent should therefore ask:
- required English level
- visa subclass
- whether the learner must meet PTE specifically or can use another accepted pathway
- test date or planned test date

## Practical intake flow for the skill
When the learner mentions immigration, use this sequence:

1. What English level do you need: functional, vocational, competent, proficient, or superior?
2. What visa subclass are you applying for?
3. Are you using PTE Academic specifically?
4. Do you already have a score report? If yes, what are your four communicative skill scores?
5. What is your application deadline or test date?
6. Re-check the official Home Affairs source pages before giving a final requirement judgement.
