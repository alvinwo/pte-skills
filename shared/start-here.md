# Start Here

If you are new to this repo, start with this page.

## Who this repo is for
This repo helps PTE learners who want practical preparation guidance rather than just random tips.

It is especially useful for learners who need:
- a student visa English result
- 4个7
- 4个8
- a prioritized study plan instead of practicing everything equally

## What this repo is best at
This repo is strongest at helping learners answer:
1. what score target am I aiming for?
2. which question types matter most for my situation?
3. what can I deprioritize?
4. what should I do every day?
5. how should I practice the most important question types?

## What this repo does not try to do yet
This repo is not yet:
- a full question bank
- a full scoring engine
- a full mock-analysis system
- a complete skill library for every single PTE task

It is a strategy-first coaching pack.
