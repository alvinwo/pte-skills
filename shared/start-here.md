# Start Here

Use this repo in four steps.

## Step 1: define the target
Ask:
- overall target
- speaking target
- writing target
- reading target
- listening target
- whether this is for a student visa, 7炸, or 8炸

## Step 2: identify the weak area
Ask the learner:
- which communicative skill feels weakest
- which question types feel weakest
- whether they have recent mock scores or real scores

If they do not know, infer it from score gaps or repeated task failures.

## Step 3: build the plan
Read:
- `shared/priority-map.md`
- `shared/study-plan.md`

Then choose:
- 2 primary tasks
- 1 support task
- 1 maintenance task

## Step 4: go to the right skill pages
For many learners, the first skill pages to open are:
- `skills/pte-repeat-sentence.md`
- `skills/pte-write-from-dictation.md`
- `skills/pte-read-aloud.md`
- `skills/pte-describe-image.md`
- `skills/pte-reading-blanks.md`

## Simple rule
Target decides the score floor.
Weak area decides the first priority.
