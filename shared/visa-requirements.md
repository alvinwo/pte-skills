# Visa Requirements

This file is a short reference for Australian immigration-related English targets.

## Important note
Validate thresholds against the Department of Home Affairs before giving final migration advice.
The saved JSON file is at:
- `data/au-home-affairs-english-requirements.json`

## Common learner language
- 7炸 = Proficient English
- 8炸 = Superior English

These are common immigration-style learner labels in this repo.

## Current saved reference
- Functional English: overall PTE at least 24
- Vocational English: L33 / R36 / W29 / S24
- Competent English: L47 / R48 / W51 / S54
- Proficient English: L58 / R59 / W69 / S76
- Superior English: L69 / R70 / W85 / S88

## How to use this file
1. ask which level the learner needs
2. ask whether they already have scores
3. plan from the weakest required section, not from the overall score alone
