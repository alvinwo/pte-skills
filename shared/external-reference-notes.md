- routine design
- template use philosophy
- review habits

## High-value ideas worth preserving
### 1. Preparation should be prioritized, not equal
Not all question types deserve equal effort.
This matches the repo's strategy-first approach.

### 2. Study order matters
A strong practical sequence is:
- listening first
- speaking next
- reading after that
- writing later

Reason:
- listening supports speaking
- speaking and listening are high-impact in PTE
- reading can be improved partly through efficient high-yield task work
- writing can often be prepared later with templates, key points, and controlled practice

### 3. WFD should start early and continue throughout prep
This strongly supports the repo's existing view that WFD is a core daily engine task.

### 4. RS improves when speaking is warmed up first
A good practical note is that RS practice works better after short speaking warm-up rather than when the mouth is still cold.

### 5. Reading review needs repeated close intervals
For reading blanks and reorder tasks, long gaps between review rounds reduce retention.
Repeated short-cycle review works better than occasional large sessions.

### 6. For many targets, writing does not need unlimited practice volume
Instead, writing prep can be efficient if the learner:
- stabilizes one safe structure
- prepares reusable key points
- controls spelling and grammar
- does enough real typed practice to avoid collapse under time pressure

### 7. DI and RL should be trained for fluent output, not overcomplicated content perfection
This aligns with the repo's guidance that structure, fluency, and relevance often matter more than overambitious content density.

### 8. SST can be treated as a high-value integrated task, but should not dominate v1 scope yet
This reinforces SST as a strong future addition for the repo.

## Guardrails
The repo should not blindly copy older tactical advice from any single post.
Before adopting an idea, check whether it still fits:
- the current official task lineup
- the repo's scenario-based coaching model
- the current score-priority logic

## Repo implications
The most useful changes inspired by these references are:
- clearer study-order guidance
- explicit speaking warm-up before RS
- stronger advice on spaced review for reading tasks
- clearer statement that WFD starts from day 1
- stronger framing that writing can be prepared efficiently rather than endlessly
