# Sample Session Outline

## Learner profile
- Target: improve overall PTE performance
- Current focus: speaking and listening-heavy tasks
- Immediate weakness: misses key details under time pressure

## Example session flow
1. Warm-up with shared speaking tips.
2. Run one Repeat Sentence prompt.
3. Give feedback on fluency and content recall.
4. Run one Describe Image prompt.
5. Give feedback on structure and delivery.
6. End with one Write From Dictation micro-drill.
7. Recommend one reading tip and one listening tip for homework.

## Example coaching style
- Start with what the learner did well.
- Identify the biggest missed opportunity.
- Give only a few next actions.
- Keep feedback easy to apply in the next attempt.
