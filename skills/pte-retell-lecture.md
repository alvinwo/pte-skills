# PTE Retell Lecture

## Purpose
- retell the main points of a lecture in clear spoken English
- support both Listening and Speaking

## When to use
- when Listening and Speaking both need support
- when RL is a repeated weak point

## Success looks like
- clear start
- enough relevant content
- simple linking language
- steady fluency

## Common mistakes
- overcomplicated template use
- weak notes
- long hesitation before speaking
- trying to reproduce everything

## Practice method
1. take simple notes
2. capture the topic and main points
3. use a short structure
4. keep transitions simple
5. favor fluency over perfection

## Daily target
- support-task volume, not the whole study plan
