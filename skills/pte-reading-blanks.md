# PTE Reading Blanks

## Purpose
- fill reading blanks using grammar, meaning, and collocation
- prioritize one of the most practical Reading tasks

## When to use
- when Reading is weak
- before spending large amounts of time on reading MCQ

## Success looks like
- strong grammar judgment
- strong collocation awareness
- good meaning control in context

## Common mistakes
- choosing by meaning only
- not checking grammar role first
- weak collocation memory
- spending too much time on one blank

## Practice method
1. inspect the blank first
2. decide the grammar role
3. check meaning in context
4. use collocation as the final filter
5. save repeated mistakes for review

## Daily target
- regular reading block when Reading is weak
