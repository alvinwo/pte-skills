# PTE Respond to a Situation

## Why it matters
RTS matters, but it should usually be learned as a safe structure task rather than a place for fancy language.

## What good performance looks like
- direct response to the situation
- clear purpose
- simple polite language
- no major hesitation

## Common mistakes
- memorizing a rigid script
- sounding unnatural
- missing the actual purpose of the task
- overcomplicating the wording

## Practice method
1. identify the situation and purpose
2. use a simple opening
3. give the main message clearly
4. add one useful detail if needed
5. end cleanly

## Daily target
- light support practice

## When to prioritize it
- after the bigger engine tasks are stable
- when the learner specifically struggles with this format
