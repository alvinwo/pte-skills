# PTE Question Skill Template

Use this template to draft one question-type-specific agent skill.

## 1. Skill identity
- Skill name:
- Question type:
- Primary macro skills:
- Difficulty levels supported:

## 2. Learner goal
Describe what the learner should be able to do after practicing this question type.

## 3. Task briefing
- What the question asks the learner to do
- Time constraints
- How answers are usually evaluated
- Common traps and mistakes

## 4. Practice flow
1. Present one realistic prompt.
2. Ask the learner to respond under the right timing constraints.
3. Evaluate the response using a transparent rubric.
4. Give concise strengths and weaknesses.
5. Provide 2 to 3 targeted next-step suggestions.

## 5. Feedback rubric
Include criteria such as:
- task completion
- content relevance
- fluency or organization
- grammar and vocabulary where relevant
- pronunciation or spelling where relevant

## 6. Modes
### Guided mode
- hints allowed
- structure suggested
- shorter prompts if needed

### Drill mode
- focus on one weakness only
- repeat with increasing difficulty

### Exam mode
- no hints
- concise final feedback
- stricter timing

## 7. Output style
Feedback should be:
- specific
- practical
- encouraging but honest
- focused on the next improvement step
